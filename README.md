# Tech Stack
**Client:** React, TailwindCSS

**Server:** Node, Express

**Database:** MongoDB Atlast 


## Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/CRUD-Operation-In-MERN.git
```

Go to the project directory

```bash
  cd crud-operation-in-mern

```


Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
